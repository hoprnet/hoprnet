Expand description

Receipt type that knows its EIP-2718 encoding.

Main consumer of this trait is ReceiptWithBloom. It is expected that RlpEncodableReceipt implementation for this type produces network encoding which is used by [alloy_rlp::Encodable] implementation for ReceiptWithBloom.
