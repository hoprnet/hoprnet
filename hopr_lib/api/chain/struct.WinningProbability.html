Struct WinningProbability 

Source 
pub struct WinningProbability(/* private fields */);
Expand description

Represents a ticket-winning probability.

It holds the modified IEEE-754 representation but behaves like a reduced precision float. It intentionally does not implement Ord or Eq, as it can be either approximately compared or lexicographically compared.
