## CT node

```sh
DEBUG=hopr* node ./lib/index.js <Private key as hex>
```

Example

```sh
DEBUG=hopr* node ./lib/index.js 0xcb1e5d91d46eb54a477a7eefec9c87a1575e3e5384d38f990f19c09aa8ddd332
```
