Macro debug_print

macro_rules! debug_print {
    ($( $args:expr ),*) => { ... };
}
Expand description

Uses the log crate to perform logging. This must be enabled using the feature flag debug-print.

§Usage

use sea_orm::debug_print;

#[derive(Debug)]
enum FooError {
    Bar,
    Baz,
}

debug_print!("{:?}", FooError::Bar);
