# hopr-protocol-hopr

Contains implementation of the HOPR protocol as specified in RFC-0004 & RFC-0005

Part of the [HOPR](https://hoprnet.org/) protocol implementation.

## Specification

- [RFC-0004: HOPR Packet Protocol](https://github.com/hoprnet/rfc/blob/main/rfcs/RFC-0004-hopr-packet-protocol/0004-hopr-packet-protocol.md)
- [RFC-0005: Proof of Relay](https://github.com/hoprnet/rfc/blob/main/rfcs/RFC-0005-proof-of-relay/0005-proof-of-relay.md)
