Struct AsyncWriteSink 

Source 
pub struct AsyncWriteSink<const C: usize, S>(pub S);
Expand description

Wraps a [futures::Sink] that accepts Box<[u8]> with an [AsyncWrite] interface, with each write to the underlying Sink being at most C bytes.
